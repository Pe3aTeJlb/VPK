Как добавить новую модель в AR:
1. Собрать презентационную модель
2. Собрать модель для игры, если такая предполагается
3. Сделать иконку для отображения в панели выбора
4. В сцене Main идём в AR Core Device -> AR Core UI -> Content View -> ViewPoint -> Content -> В инспекторе ContentController Добавляем в models модель 
5. В модели -> Float UI Controller добавить модель для игры

Как собрать презентационную модель:
1. Создать пустышку
2. Добавить к пустышке модель дочерним объектом
3. Добавить Float UI System дочерним объектом

(Опционально)
4. К Float UI System добавить Float Annotation
5. Добавить в модель!!!! Float UI System TargetPoint
6. Связать пункты 4 и 5 дабавив в Annotations новым пунктом (см инспектор Float UI System)

7. Для поворота, уменьшения/увеличения модели, в Float UI System присвоить полю Model нашу модель из иерархии
8. Если есть игра, то в Float UI System присвоить полю Model префаб модели для игры.

Остальное не трогаем.

Как собрать модель для игры
Подготовка модели
1. Модель после импортирования в юнити должна распологаться вдоль оси Z.
2. Колёса должны быть отдельными мешами изначально
3. Колёса обязательно должны крутиться по оси Х, иначе будет шляпа.

Сборка.
1. Создать пустышку, добавить к ней нашу модель
1.1 Добавить пустышку к корню(пустышке), расположить в предполагаемом центре тяжести.
2. В иерархии, открываем модель, находим колёса, добавляем к ним пустышки.
3. Меняем колёса и пустышки пестами, т.е. теперь пустышки стали родительскими по отношению к колёсам.
4. Добавляем пустышкам whellcollider, подгоняем под меш колеса.
5. В корень игровой модели(та что пустшка самого верхнего уровня) добавляем по очерёдности: Rigidbody, SimpleCarController, SteeringWheel;
6. Сразу говорю, SteeringWheel мы больше не трогаем. В Rigidbody назначаем вес.
7. Что я могу сказать про SimpleCarController. Он относительно кривой. Обязательно нужно назначить Center of mass (geynr 1.1), 
GearRation - передаточное число (количество оборотов руля/макс угол поворота колёс), MaxSteerAngel и MaxVolantAngel - угол поворота колёс, ставить одинаковые значения
В AxelInfo добавить колёса, там всё понятно.
С остальным надо эксперементировать.
