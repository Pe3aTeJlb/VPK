Как добавить новую модель в AR:
1. Собрать презентационную модель
2. Собрать модель для игры, если такая предполагается
3. Сделать иконку для отображения в панели выбора
4. В сцене Main идём в AR Core Device -> AR Core UI -> Content View -> ViewPoint -> Content -> В инспекторе ContentController Добавляем в models Вышеперечисленное

Как собрать презентационную модель:
1. Создать пустышку
2. Добавить к пустышке модель отдельным объектом
3. Добавить Float UI System отдельным объектом

(Опционально)
4. К Float UI System добавить Float Annotation
5. Добавить в модель!!!! Float UI System TargetPoint
6. Связать пункты 4 и 5 дабавив в Annotations новым пунктом (см инспектор Float UI System)
